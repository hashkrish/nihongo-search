# Nihongo Search

Instant search of japanese words by meaning, kanji, pronounciation.

## Dependencies

- Go 1.22

## Usage

1. Clone the repository

``` shell
git clone https://github.com/hashkrish/nihongo-search.git
```

2. Run the server

``` shell
cd nihongo-search
go run main.go
```

The server will start at `localhost:8080` with minimal UI.
